# 章节示例

示例代码按章节拆分。每章练习只改自己的 crate，避免互相干扰。

## 使用规则

- 只验证当前章节：`cargo test -p chXX-name`
- 运行第 1 章可执行示例：`cargo run -p ch01-rust-program-shape --bin variables`
- 全仓库验证：从仓库根目录跑 `cargo test`
